A lighter way to see your Git history

Git Light is a GitKraken-inspired desktop client with a commit graph, branch sidebar, working tree panel, and keyboard-driven navigation — built with Electron, Vue 3, and TypeScript.

Everything you need to navigate a repo

A focused Git UI shell with room to plug in a real backend.

⎇

Commit graph

Virtualized SVG lanes with zoom, auto-scroll, and arrow-key selection across history.

⑂

Branch sidebar

Favorites, groups, remotes, tags, stashes, and worktrees — all searchable from one panel.

±

Working tree

Stage and unstage files, resolve conflicts, and compose commits with sign-off support.

⌘

Keyboard shortcuts

Navigate commits, focus search, and drive the UI without leaving the keyboard.

⚙

Pluggable backend

GitService interface separates UI from Git operations — mock included for development.

◆

Dark-first design

Dense, professional aesthetic tuned for long sessions — inspired by GitKraken's visual language.

Built with modern tools

Electron desktop shell, Vue 3 renderer, and a typed Git service layer.

Get started

Clone the repo, install dependencies, and launch in development mode.

terminal
# Clone and run
git clone https://github.com/ftuyama/git-light.git
cd git-light
npm install
npm run dev
